# ユーザー系API

### プロフィールの取得

##### Endpoint

```
GET /api/v1/user/profile
```

### フォローの取得

##### Endpoint

```
GET /api/v1/user/following
```

### フォロワーの取得

##### Endpoint

```
GET /api/v1/user/followed
```

### フォローする

##### Endpoint

```
GET /api/v1/user/follow
```

### フォローをはずす

##### Endpoint

```
GET /api/v1/user/unfollow
```

### プロフィールを編集する

##### Endpoint

```
GET /api/v1/user/profile/edit
```
